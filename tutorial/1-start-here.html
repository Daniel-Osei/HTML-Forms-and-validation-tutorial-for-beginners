This tutorial will teach you how to build and validate HTML Forms

What we will learn during ths tutorial?

  1. How to create HTML Forms
  2. How to submit / reset form
  3. What are different input types
  4. How to perform validation

How to start?

1. Prepare the environment

You can check this tutorial on how to setup environment.

1.1. Text editor

Download and install Visual Studio Code

1.2 Browser

Download and install one of the following browser:

  • Google Chrome
  • Firefox
  • Opera

1.3 Lesson files

Download or clone lesson files into your computer.

You will see the following files

        tutorial/ 
            1-start-here.html (current file)
            2-forms.html
            3-input-types.html
            4-html-validation.html
            5-js-validation.html
        final/
            form.html
            results.html

2. Start learning

Navigate to tutorial folder and follow lesson files in given order:

  1. 1-start-here.html (current file)
  2. 2-forms.html
  3. 3-input-types.html
  4. 4-html-validation.html
  5. 5-js-validation.html
